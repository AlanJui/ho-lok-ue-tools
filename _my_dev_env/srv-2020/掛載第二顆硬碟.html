      # 掛載第二顆硬碟作業指引

      令作業系統於開機時，自動掛載第二顆台硬碟。

      # 電腦內硬碟摘要

      • 第一顆硬碟（500GB)： /dev/sda
      • 第二顆硬碟（2TB)： /dev/sdb

      # 作業程序

      1. 確認作業系統偵測到兩顆內接硬碟皆已存在：
      $ ls /dev/sd*
      1
      1. 驗證第二顆硬碟的容量
      $ sudo fdisk -l /dev/sdb
      1
      1. 查詢第二顆硬碟的 UUID 查詢 /dev/sdb 的 UUID
      $ sudo blkid
      1

      1. 編輯「硬碟自動掛載設定檔」 /etc/fstab

        4.1 開啟設定檔

        $ sudo vim /etc/fstab
        1

        4.2 加入設定

        # External Disk
UUID=9279ede9-49f6-44fe-b481-261500f67675 /NAS ext4 defaults 0 2
        1
        2

        設定檔內完整的設定：

        # /etc/fstab: static file system information.

      2. 設定掛載點 設定第二顆硬碟（/dev/sdb），掛載點位於目錄路徑： /NAS。

      $ sudo mkdir /NAS
      1
      1. 第一次手動掛載硬碟
      $ sudo mount /NAS
      1
      1. 驗證已成功掛載
      $ sudo df -h
      1
