# 安裝Heroku工具

輸入安裝指令（官方文件說要先安裝Ruby，但在Ubuntu 16.04沒做過安裝動作）：

wget -O- https://toolbelt.heroku.com/install-ubuntu.sh | sh

驗證完成安裝：

heroku --version
