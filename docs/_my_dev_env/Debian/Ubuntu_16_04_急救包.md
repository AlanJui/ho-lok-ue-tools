# Ubuntu 16.04 急救包


# 無法正常 Login

 1. 完成開機，看到 Login 晝面時，切換到 Terminal Mode。
 
 **切換到 Terminal Mode：**

    <Ctrl> + <Alt> + <F1>


**切換到 GUI Mode：**

    <Ctrl> + <Alt> + <F7>


----------
# 無法正常開機～無法正常完成 Login

需要進入 Recovery Mode ，檢查問題狀況．

１. 在 BIOS Booting 畫面剛顯示時，立即按＜Esc＞鍵．

 2. 自選單選擇 Recovery Mode ．
